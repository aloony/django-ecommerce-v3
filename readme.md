#### 1 - Clone repo

```
git clone https://github.com/aloony/django-ecommerce-v3
```

#### 2 - Install requirements

```
pip install -r requirements.txt

```

#### 3 - Make DB

```
python manage.py makemigrations
python manage.py migrate
```

#### 4 - Start server

```
python manage.py runserver
```
